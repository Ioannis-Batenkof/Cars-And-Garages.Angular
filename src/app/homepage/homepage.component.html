What is this place?...


This is Ioannis' first ever Angular Project. It is initially based on the Backend he created. Its theme.. well, you guessed it: Car and Garages.

You create cars and assign them to your garage to claim ownership. If a mistake occurs, you can simply edit the details for both cars and garages.

This program needs to be 100% reconfigured. So it is marked as Development Environment



How it Works..


Here are the basic Should-Know stuff:

  • You can change EVERYTHING, except the ID (It is generated automatically)
  • The same car cannot be assigned to 2 or more garages. (That is physically not possible)
  • If a car is already assigned to another garage it will be inaccessible to all other garages
